# Sprites Collection Tracker v1.28

Created/Developed by Happy Dude Customs x GBGamerBros

A mobile-friendly collection tracker packaged for GitHub Pages.

## Publish

1. Upload every file in this folder to the root of a public GitHub repository.
2. Open **Settings → Pages**.
3. Choose **Deploy from a branch**, select **main** and **/(root)**, then save.

## Privacy

The app contains no analytics, tracking, geolocation, account login, email address, phone number, IP address, or external network request. Collection progress is stored only in the visitor's browser using local storage.
